This is a puzzle platformer game where you two Players control two Martians. They don't
know how they got to the planet they are on now but, with their rock manipulation powers, they are
able to navigate through and figure out where they are.

Controls:<br>
Blue Martian (Kiroh)<br>
W = Jump<br>
A = Left<br>
D = Right<br>
E = Interact<br>
Q = Drop items in Inventory<br>    

Red Martian (Bryx)<br>
Up Arrow = Jump<br>
Left Arrow = Left<br>
Right Arrow = Right<br>
CTRL = Interact<br>
Shift = Build Floor<br>
